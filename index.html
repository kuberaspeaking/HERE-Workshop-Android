Scope

In this workshop we will set up a simple map using the HERE SDK for Android Explore Edition in 5 simple steps.

  1. Download the HERE SDK
  2. Import SDK into your example
  3. Load example and enter credentials
  4. Build and Run
  5. Final Result

Pre-requisites

Get Freemium

To access any of the APIs or SDKs, first get your credentials by signing up for a freemium account.

Get Android SDK Access Key and Secret

  • Go to the project page on your account at developer.here.com.
  • Scroll down to the HERE SDK for Android or iOS section and click on Create credentials.


  • Download the file which contains the ACCESS KEY ID and ACCESS KEY SECRET

Install Android Studio

  • Download the Android Studio
  • Install the studio by following the video below. Start at 01:27 and follow until 05:46.

Join our Slack

During the workshop, if you have any questions, please drop them on our Slack channel weu-workshop

Let's begin!

STEP 1

Download HERE SDK for Android Explore Edition

  • Go back to the project page on your account at developer.here.com.
  • Scroll down to the HERE SDK for Android or iOS section and click on next to Download SDK for Android (Explore Edition).


  • Extract the downloaded file in your workshop folder

STEP 2

Import SDK

  • Navigate to the '*.aar' file in the extracted folder and copy it
    • heresdk-explore-android-4.3.3.0.2053
    • heresdk-explore-android-4.3.3.0.2053.aar
  • Paste this file in the libs folder of the HelloMap example.
    • heresdk-explore-android-4.3.3.0.2053
    • heresdk-explore-android-examples-4.3.3.0
    • HelloMap
    • app
    • libs

STEP 3

Load Example

  • Open the Android Studio and click on Open an existing Android Studio project
  • Navigate to the extracted folder in the previous step to open the example HelloMap
    • heresdk-explore-android-4.3.3.0.2053
    • heresdk-explore-android-examples-4.3.3.0
    • HelloMap


Enter Credentials

  • Click on the Android icon on the top-left of the IDE.
  • Open the file AndroidManifest.xml
    • app
    • manifests
    • AndroidManifest.xml


  • In the code window, go to the section Set your credentials for the HERE SDK.
  • From the downloaded credential file, copy the Access Key ID and the Access Key Secret and paste it in the code.


STEP 4

Start Emulator

You need to do this step only once when you start the Android Studio.

  • In the tool bar, open the AVD Manager


  • Click on the green arrow to launch the emulator.
  • A phone emulator will appear on your screen. Click on the power-on button on the phone.


Build and Run

  • Click on Build in the menu and then click on Make Project. Wait until the build is complete.
  • Click on Run in the menu and then click on Run App.
  • Your app will load on the phone emulator.
  • Allow the permissions that the app needs.

STEP 5

Final Result

Your app should look something like this. You can scroll around just like a normal app.
